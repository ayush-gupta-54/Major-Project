# 🌐 Semantic Codebase Cartography

A Streamlit-based tool that analyzes Python code and visualizes function relationships using graph structures.

---

## 🚀 Features

* Upload Python source files
* Extract functions and call relationships
* Visualize code as an interactive graph
* Display structural and semantic insights

---

## 🛠 Tech Stack

* Python
* Streamlit
* NetworkX
* PyVis
* PyTorch
* NumPy

---

## ▶️ How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
streamlit run app.py
```

3. Open in browser:

```
http://localhost:8501
```

---

## 📁 Project Structure

* `app.py` → Main UI (Streamlit)
* `parser.py` → Code parsing logic
* `graph.py` → Graph creation
* `model.py` → ML / embeddings
